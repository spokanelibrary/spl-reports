Naturalization Index import

Load a tab-delimited file into the database for searching.
Reset


Summary

You MUST:
  • Upload a tab-delimited txt file named naturalization.txt
  • You can save as "tab delimted text" from Excel
  • Have the following (and only the following) columns in the following order:
    • Roll #
    • Item #
    • Document #
    • Name
    • Birthdate
    • Location of Birth
    • Document Year

Note: The first column is ignored, on the assumption that is a header row. The actual header names are not important, so long as the contain the information in the order listed above.
